# Weat API

Weat API Service

## Getting Started

1. Clone repository

   ```shell
   git clone https://github.com/wintery-software/weat-api.git
   ```

2. Install dependencies

   ```shell
   bundle install
   ```

3. Prepare database

   ```shell
   rails db:create
   rails db:migrate
   rails db:seed
   ```

4. Start server

   ```shell
   rails server
   ```
